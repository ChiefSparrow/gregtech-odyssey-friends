GREGTECH ODYSSEY — FRIENDS EDITION
ПРОСТАЯ УСТАНОВКА ДЛЯ TLAUNCHER

1. Установите TLauncher и запустите через него любую версию Minecraft один раз.
2. Полностью распакуйте этот ZIP.
3. Дважды нажмите INSTALL-GTO-TLAUNCHER.bat.
4. Нажмите «Установить сборку» и дождитесь сообщения об успешной проверке.
5. Установщик сам добавит точный локальный Forge и покажет игровую папку.
6. Полностью перезапустите TLauncher.
7. В TLauncher выберите:
   локальную версию 1.20.1-forge-47.4.20
   НЕ удалённую запись «Forge 1.20.1»
   Java по умолчанию (профиль сам выберет установленную Java 21)
   НЕ Java 8 и НЕ Java 17
   RAM 8192–12288 МБ (рекомендуется 10240 МБ)
   «Принудительное обновление» выключено

Prism Launcher не нужен.

Установщик:
- не требует прав администратора;
- не изменяет реестр и системные папки;
- не использует PowerShell;
- не отключает антивирус и не просит добавлять исключения;
- скачивает сторонние моды по HTTPS с официального CDN CurseForge;
- скачивает официальный Forge 47.4.20 с maven.minecraftforge.net;
- проверяет наличие Java 21 и при необходимости скачивает официальный
  Eclipse Temurin JRE 21.0.11+10 x64;
- устанавливает Java 21 приватно в игровую папку, без прав администратора;
- проверяет Forge по SHA-256 до запуска его официального установщика;
- проверяет размер и SHA-256 каждой загрузки и всех 177 обязательных файлов;
- проверяет готовый локальный профиль и критические файлы Forge;
- закрепляет за профилем Java 21 и проверяет файлы приватной Java;
- ставит режим прогрессии GTO Easy, оставляя ванильную сложность Normal;
- прекращает установку при любом несовпадении.

В архиве нет самодельного EXE. Исходный код Java-установщика, полный список URL
и контрольных сумм находятся в папке SOURCE.

Если антивирус именно обнаружил угрозу, не отключайте его. Удалите архив,
скачайте его повторно только из GitHub Releases и сверьте SHA-256 с файлом
.sha256 рядом с архивом.

Если в обычной .minecraft уже есть другая сборка, установщик автоматически
предложит отдельную папку .gto-friends-1.0.4. Укажите именно её в настройках
TLauncher как игровую папку.

Если уже установлена версия 1.0.2 или 1.0.3, новый установщик найдёт её и
обновит конфигурацию, Java 21 и локальный Forge без повторного скачивания
всех модов.

Мир GTO Easy установщик сохранит. Если в папке найден мир GTO Normal/Expert,
обновление остановится до изменения папки: такой мир нельзя автоматически
понизить до Easy, поэтому для Easy-сервера нужна отдельная чистая игровая папка.

Не выбирайте удалённую запись «Forge 1.20.1»: в TLauncher 2.9331 она содержит
ошибочные метаданные и падает до запуска Minecraft. Нужна именно локальная
версия 1.20.1-forge-47.4.20, которую создаёт этот установщик.

Игрок без лицензии сможет войти только на сервер, специально настроенный для
offline-аккаунтов. Такой сервер обязательно должен иметь отдельную систему
регистрации и защиту ников.
