GREGTECH ODYSSEY — FRIENDS EDITION
ПРОСТАЯ УСТАНОВКА ДЛЯ TLAUNCHER

1. Установите TLauncher и запустите через него любую версию Minecraft один раз.
2. Полностью распакуйте этот ZIP.
3. Дважды нажмите INSTALL-GTO-TLAUNCHER.bat.
4. Нажмите «Установить сборку» и дождитесь сообщения об успешной проверке.
5. Установщик покажет готовую игровую папку.
6. В TLauncher выберите:
   Minecraft 1.20.1
   Forge 47.4.20 БЕЗ значка TL
   Java 17
   RAM 8192–12288 МБ (рекомендуется 10240 МБ)

Prism Launcher не нужен.

Установщик:
- не требует прав администратора;
- не изменяет реестр и системные папки;
- не использует PowerShell;
- не отключает антивирус и не просит добавлять исключения;
- скачивает сторонние моды по HTTPS с официального CDN CurseForge;
- проверяет размер и SHA-256 каждой загрузки и всех 177 обязательных файлов;
- прекращает установку при любом несовпадении.

В архиве нет самодельного EXE. Исходный код Java-установщика, полный список URL
и контрольных сумм находятся в папке SOURCE.

Если антивирус именно обнаружил угрозу, не отключайте его. Удалите архив,
скачайте его повторно только из GitHub Releases и сверьте SHA-256 с файлом
.sha256 рядом с архивом.

Если в обычной .minecraft уже есть другая сборка, установщик автоматически
предложит отдельную папку .gto-friends-1.0.2. Укажите именно её в настройках
TLauncher как игровую папку.

Не выбирайте версию со значком TL: система скинов может добавить лишний
TLSkinCape в папку mods. Если точной Forge 47.4.20 без значка TL в списке нет,
не выбирайте другую версию — сообщите владельцу сборки.

Игрок без лицензии сможет войти только на сервер, специально настроенный для
offline-аккаунтов. Такой сервер обязательно должен иметь отдельную систему
регистрации и защиту ников.
