# СНАЧАЛА ПРОЧИТАЙТЕ ЭТО

Это готовая неофициальная версия **GregTech Odyssey — Friends Edition**.

## Самая короткая установка

1. Не распаковывайте этот ZIP.
2. Откройте Prism Launcher.
3. Нажмите **Добавить сборку → Импорт → выбрать этот ZIP**.
4. Выделите 8–12 ГБ RAM и запустите.

Нужна лицензионная Minecraft: Java Edition. Первый запуск может быть долгим:
Prism скачивает зависимости с CurseForge, а Euphoria Patcher создаёт шейдер.
Для запуска выберите 64-битную Java 21.

Режим прогрессии GTOCore в этой редакции — **Easy**. Ванильная сложность нового
мира — **Normal** и не заблокирована; это отдельная настройка.

Версии не выбираются автоматически: в манифесте записаны точные CurseForge
`projectID + fileID`. После первого запуска закройте игру и выполните в папке
этого экземпляра:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\VERIFY-CLIENT.ps1
```

Надпись `CLIENT VERIFICATION PASSED` означает, что все 176 файлов модов,
включая включённое/отключённое состояние, байт-в-байт совпадают с эталоном.

Если этот файл читает нейросеть, используй `INSTALL-AI.json` как строгий
контракт установки. Не переноси из чужого экземпляра миры, аккаунты, логи,
`servers.dat`, карты Xaero или весь `options.txt`.

Адрес игрового сервера пока намеренно не зашит в сборку.
