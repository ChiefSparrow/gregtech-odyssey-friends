GREGTECH ODYSSEY — FRIENDS EDITION 1.0.4
ЛИЦЕНЗИЯ MINECRAFT / PORTABLE PRISM LAUNCHER

БЫСТРАЯ УСТАНОВКА

1. Полностью распакуйте скачанный ZIP в обычную папку.
2. Закройте Minecraft и все окна Prism Launcher.
3. Дважды нажмите INSTALL-GTO-LICENSED.bat.
4. Дождитесь сообщения об успешной установке.
5. В открывшемся Prism Launcher добавьте учётную запись Microsoft:
   справа сверху нажмите на кнопку аккаунта -> Manage Accounts -> Add Microsoft.
6. Выберите GregTech Odyssey — Friends Edition и нажмите Launch.

Ручной импорт сборки, выбор Forge и настройка Java не нужны.

ЧТО УСТАНАВЛИВАЕТСЯ

- официальный portable Prism Launcher 11.0.3;
- точная Friends Edition 1.0.4 со всеми закреплёнными модами;
- фикс дюпа Simple Crafting Terminal;
- исправления Easy Villagers, сбор растений ПКМ и баланс тыкв/арбузов;
- русские переводы, исправления квестбука, шейдеры и падающие деревья;
- приватная 64-битная Java 21;
- Minecraft 1.20.1 и Forge 47.4.20;
- режим прогрессии GTO Easy при ванильной сложности Normal.

По умолчанию всё хранится отдельно:

%LOCALAPPDATA%\GTO-Friends-Licensed

Существующие Minecraft, Prism Launcher и их аккаунты не изменяются.
После установки эту сборку можно запускать файлом PLAY-GTO-LICENSED.bat
из указанной папки. Сам лаунчер находится по пути
PrismLauncher\prismlauncher.exe.

ОБНОВЛЕНИЕ

Для проверки или восстановления версии 1.0.4 повторно запустите её
INSTALL-GTO-LICENSED.bat и укажите прежнюю папку. Установщик сохранит аккаунт
Microsoft, настройки Prism и миры GTO Easy.
Если найден мир GTO Normal или Expert, обновление остановится до изменения
файлов и попросит выбрать новую пустую папку.

Переход на будущую версию выполняйте в прежней папке только тогда, когда в
описании нового релиза прямо указана совместимость обновления с 1.0.4.
Иначе установите новую версию отдельно: установщик не станет молча смешивать
разные наборы модов.

БЕЗОПАСНОСТЬ И ПРИВАТНОСТЬ

Установщик не требует прав администратора, не меняет реестр и не читает
accounts.json. Авторизация выполняется только самим официальным Prism Launcher.
Загрузка Prism, файлы сборки и вложенная Java проверяются по закреплённым
размерам и хэшам.
