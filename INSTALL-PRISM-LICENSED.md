# Установка с лицензией Minecraft — Prism Launcher

Это основной и рекомендуемый способ установки **GregTech Odyssey — Friends
Edition**.

## Что понадобится

- купленная Minecraft: Java Edition;
- [Prism Launcher](https://prismlauncher.org/download/) версии 11.0.3 или новее;
- 64-битная Java 21;
- 8–12 ГБ свободной оперативной памяти для игры;
- несколько гигабайт свободного места на диске.

## Установка

1. Откройте страницу
   [последнего релиза](https://github.com/ChiefSparrow/gregtech-odyssey-friends/releases/latest).
2. Скачайте файл вида
   `GregTech-Odyssey-Friends-0.5.6-v*.zip`.
   Не скачивайте `Source code.zip`.
3. Не распаковывайте архив.
4. В Prism Launcher нажмите
   **Добавить сборку → Импорт → выбрать скачанный ZIP**.
5. Дождитесь окончания загрузки всех зависимостей. Prism использует
   зафиксированные в манифесте `projectID + fileID`, а не последние версии
   модов.
6. Откройте **Настройки → Java**:
   - версия Java — `21` (64-bit);
   - минимальная память — не более `1024 MiB`;
   - максимальная память — `8192–12288 MiB`;
   - рекомендуемое значение — `10240 MiB`.
7. Добавьте Microsoft-аккаунт, на котором куплена Minecraft: Java Edition.
8. Запустите сборку. Первый запуск может быть заметно дольше обычного:
   Euphoria Patcher подготавливает шейдер.

В этой версии режим прогрессии GTOCore заранее установлен на **Easy**.
Ванильная сложность нового мира при этом остаётся **Normal** и не блокируется:
это две независимые настройки.

## Обязательная проверка перед сервером

1. Закройте Minecraft.
2. В Prism нажмите ПКМ по сборке → **Папка Minecraft**.
3. В открывшейся папке запустите PowerShell и выполните:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\VERIFY-CLIENT.ps1
```

Подключаться к серверу можно только после сообщения:

```text
CLIENT VERIFICATION PASSED
```

Проверяются 176 файлов модов, включая 15 намеренно отключённых, и два
обязательных OpenLoader-ресурса. Изменённый, отсутствующий или лишний JAR
считается ошибкой.

Если проверка не прошла, не обновляйте моды вручную. Удалите неудачно
импортированный экземпляр и снова импортируйте релизный ZIP как новую сборку.

Не переносите сюда мир, созданный в режиме GTO `Normal` или `Expert`: режим
GTO записан в `level.dat` и штатно не понижается до `Easy`.

## Шейдер

Если шейдер не включился автоматически:

1. откройте **Настройки → Графика → Наборы шейдеров**;
2. выберите
   `ComplementaryReimagined_r5.6.1 + EuphoriaPatches_1.7.7`;
3. для слабого компьютера уменьшите профиль качества в настройках шейдера.

## Обновления сборки

Новую версию всегда импортируйте в Prism как отдельный экземпляр. Не нажимайте
массовое обновление модов и не заменяйте папку старой сборки целиком.

Для сетевой игры с лицензированными аккаунтами сервер следует оставлять в
безопасном режиме `online-mode=true`.
