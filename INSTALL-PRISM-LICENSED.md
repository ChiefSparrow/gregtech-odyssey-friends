# Установка с лицензией Minecraft — один ZIP с Prism Launcher

Это основной и самый простой способ установить **GregTech Odyssey — Friends
Edition 1.0.4**. Отдельно скачивать Prism Launcher, Java или Forge, вручную
создавать экземпляр и импортировать модпак не требуется.

## Что понадобится

- купленная Minecraft: Java Edition;
- 64-битная Windows 10 или Windows 11;
- подключение к интернету во время первой установки;
- не менее 8 ГБ свободной оперативной памяти для игры; рекомендуется 10 ГБ;
- несколько гигабайт свободного места на диске.

## Установка

1. Откройте страницу
   [последнего релиза](https://github.com/ChiefSparrow/gregtech-odyssey-friends/releases/latest).
2. Скачайте
   [`GTO-Friends-Licensed-Prism-Installer-v1.0.4.zip`](https://github.com/ChiefSparrow/gregtech-odyssey-friends/releases/latest/download/GTO-Friends-Licensed-Prism-Installer-v1.0.4.zip).
   Не скачивайте `Source code.zip`.
3. Полностью распакуйте ZIP в обычную папку. Не запускайте установщик прямо
   из окна архиватора.
4. Закройте Minecraft и все окна Prism Launcher.
5. Дважды нажмите `INSTALL-GTO-LICENSED.bat` и дождитесь сообщения об успешной
   установке.
6. В открывшемся Prism Launcher добавьте Microsoft-аккаунт, на котором куплена
   Minecraft: Java Edition:
   **кнопка аккаунта справа сверху → Manage Accounts → Add Microsoft**.
7. Выберите **GregTech Odyssey — Friends Edition** и нажмите
   **Launch / Запустить**.

Первый запуск может занять больше времени обычного. Ручной импорт сборки, выбор
Forge, поиск Java и настройка памяти не нужны.

## Что именно устанавливается

- официальный portable Prism Launcher 11.0.3, сборка MinGW x64;
- приватная 64-битная Eclipse Temurin Java 21;
- Minecraft 1.20.1 и Forge 47.4.20;
- точная Friends Edition 1.0.4 со всеми закреплёнными модами и ресурсами;
- исправление дюпа Simple Crafting Terminal с переносимым рюкзаком;
- исправления Easy Villagers, сбор зрелых растений ПКМ и баланс
  тыкв/арбузов;
- русские переводы и согласованные с ними исправления квестбука;
- шейдеры, TreeChop и физическое падение деревьев.

Режим прогрессии GTOCore заранее установлен на **Easy**. Ванильная сложность
нового мира остаётся **Normal** и не блокируется — это две независимые
настройки.

## Где находятся файлы

По умолчанию отдельный Prism, сборка и её Java устанавливаются в:

```text
%LOCALAPPDATA%\GTO-Friends-Licensed
```

Для следующих запусков можно использовать находящийся там файл
`PLAY-GTO-LICENSED.bat`. Обычная установка Prism, другие сборки Minecraft и
их аккаунты не изменяются.

## Как обновить сборку

Чтобы проверить или восстановить текущую 1.0.4, повторно запустите её
`INSTALL-GTO-LICENSED.bat` и укажите прежнюю папку установки.

Для перехода на будущую версию используйте прежнюю папку только тогда, когда
в описании нового релиза прямо указана совместимость обновления с 1.0.4.
Иначе установите новую версию в отдельную папку: установщик не станет молча
смешивать разные наборы модов.

Проверка и совместимое обновление сохраняют Microsoft-аккаунт, настройки Prism, одиночные миры,
`servers.dat`, скриншоты и карты Xaero. Установщик не читает и не копирует
`accounts.json`: авторизацию выполняет только официальный Prism Launcher.

Мир, созданный в режиме GTO `Normal` или `Expert`, не переводится в `Easy`
молча. Если такой мир найден, обновление остановится до изменения файлов и
предложит установить эту редакцию в отдельную чистую папку. Уже созданные миры
GTO Easy сохраняются.

## Проверка и типовые проблемы

- Если Windows показывает, что файл не найден, убедитесь, что ZIP распакован
  полностью, а `INSTALL-GTO-LICENSED.bat` лежит рядом с остальными файлами
  установщика.
- Если установщик просит закрыть Prism или Minecraft, закройте их и повторите
  запуск.
- Не обновляйте отдельные моды кнопкой массового обновления Prism: сервер
  ожидает точные версии Friends Edition 1.0.4.
- Если проверка обнаружила отсутствующий, изменённый или лишний мод, верните
  исходный состав сборки и снова запустите установщик.

Установщик не требует прав администратора и не меняет реестр. Prism, Java и
игровые файлы сверяются с закреплёнными размерами и SHA-256 до запуска.

Для сетевой игры только с лицензионными аккаунтами сервер рекомендуется
оставлять в безопасном режиме `online-mode=true`.

Официальные источники Prism Launcher и Eclipse Temurin перечислены в
[`THIRD-PARTY.md`](THIRD-PARTY.md).
