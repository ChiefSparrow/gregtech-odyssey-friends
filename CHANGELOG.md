# История изменений

## 1.0.4 — 2026-07-23

- Добавлен отдельный one-click установщик для владельцев лицензии Minecraft:
  один ZIP устанавливает официальный portable Prism Launcher 11.0.3 MinGW x64,
  приватную Temurin Java 21 и точную Friends Edition 1.0.4.
- После установки остаётся только войти в Microsoft-аккаунт внутри Prism и
  нажать **Launch**; ручной импорт модпака, выбор Forge и настройка Java больше
  не требуются.
- Повторный запуск лицензированного установщика проверяет и восстанавливает
  1.0.4 на месте, сохраняя аккаунт Prism, настройки, миры GTO Easy, список
  серверов, скриншоты и карты Xaero. Переход между версиями разрешается только
  после явной проверки совместимости; миры GTO Normal/Expert не понижаются
  до Easy молча.
- Транзакционное обновление сохраняет отдельную резервную копию, если
  автоматический откат не смог завершиться из-за блокировки файла или ошибки
  диска, и сообщает точный путь для восстановления.
- Исправлен режим сборки: GTOCore теперь заранее настроен на `Easy`, а
  независимая ванильная сложность нового мира остаётся `Normal`.
- Обновление без миров меняет только `gamePlay.difficulty`, сохраняя остальные
  настройки. Мир GTO Easy сохраняется, а при найденном мире Normal/Expert
  установщик отказывается от обновления и просит отдельную чистую папку.
- Исправлен запуск GTOCore, собранного под Java 21: локальный Forge-профиль
  теперь явно требует `java-runtime-delta` / Java 21.
- Установщик использует уже запущенную доверенную Java 21 либо скачивает
  зафиксированный Eclipse Temurin JRE 21.0.11+10 x64, проверяет SHA-256 и
  устанавливает без прав администратора внутрь игровой папки.
- Установки `1.0.2` и `1.0.3` обновляются на месте без повторной загрузки
  модов.

## 1.0.3 — 2026-07-23

- Исправлен сбой TLauncher 2.9331 в удалённой записи `Forge 1.20.1`, вызванный
  несовместимым полем `arguments.default_user_jvm`.
- Установщик теперь сам создаёт официальный локальный профиль
  `1.20.1-forge-47.4.20`; выбирать удалённый профиль TLauncher больше не нужно.
- Официальный Forge и описание Minecraft загружаются только по HTTPS и
  проверяются по зафиксированным размеру и SHA-256.
- Готовый профиль, vanilla client JAR, patched Forge client JAR и universal JAR
  проверяются до копирования и после установки.
- Установка `1.0.2` чинится на месте без повторной загрузки всех модов.

## 1.0.2 — 2026-07-23

- Исправлен запуск `INSTALL-GTO-TLAUNCHER.bat` в Windows: файл теперь
  упаковывается как ASCII без BOM с обязательными переносами CRLF.
- Убран нестабильный разбор многобайтовой кириллицы внутри `cmd.exe`, который
  мог превращать части сообщений и команд в отдельные команды.
- Добавлен реальный smoke-тест BAT из пути с пробелами, кириллицей и `!`;
  проверка релиза также запрещает BOM, не-ASCII байты и переносы LF-only.

## 1.0.1 — 2026-07-23

- Добавлен отдельный Java-установщик для TLauncher: Prism больше не нужен,
  сторонние файлы загружаются с официального CDN и проверяются по SHA-256.
- Установщик не требует прав администратора, не меняет реестр и не использует
  PowerShell на компьютере игрока; исходный код включён в архив.
- Добавлен криптографический lock-файл для 176 файлов модов и двух обязательных
  OpenLoader-ресурсов рабочего клиента.
- Добавлена строгая проверка размера, SHA-256, отсутствующих и лишних модов.
- Проверяется также включённое/отключённое состояние опциональных модов.
- Уточнено, что CurseForge-манифест использует точные `projectID + fileID`, а
  не автоматически выбранные последние версии.

## 1.0.0 — 2026-07-20

- Добавлен фикс дюпа Simple Crafting Terminal и переносимого рюкзака.
- Добавлен фикс Easy Villagers и сбор зрелых культур ПКМ.
- Скорость производства тыкв и арбузов приведена к полному циклу роста стебля.
- Добавлено 1 563 русских перевода и проверка ссылок квестбука.
- Добавлены TreeChop и Panda's Falling Trees через официальный манифест.
- Добавлены Oculus, Complementary r5.6.1 и Euphoria Patches 1.7.7 через
  официальный манифест; перенесён только пользовательский пресет настроек.
- Добавлены воспроизводимая сборка, контрольные суммы и проверка приватности.
