# Установка без лицензии — запуск через TLauncher

> [!WARNING]
> TLauncher не является официальным лаунчером Mojang/Microsoft и не
> поддерживается авторами этой редакции. Рекомендуемый вариант — приобрести
> Minecraft: Java Edition и использовать Prism Launcher. Пользователь сам
> отвечает за законность и безопасность применения стороннего лаунчера.

## Важное ограничение

Релиз Friends Edition является CurseForge-пакетом. TLauncher не умеет напрямую
импортировать его `manifest.json`, поэтому нельзя просто выбрать наш ZIP в
`TL MODS`, распаковать его в общую `.minecraft` или установить похожие моды из
каталога TLauncher. В каталоге могут оказаться другие версии, несовместимые с
сервером.

Рабочая схема выглядит так:

1. Prism Launcher скачивает строго зафиксированные файлы сборки.
2. Встроенный скрипт проверяет их по SHA-256.
3. Проверенный чистый экземпляр копируется в отдельный профиль TLauncher.
4. Игра запускается через TLauncher.

Prism на первом этапе используется только как установщик сборки. Запускать
Minecraft через него и добавлять Microsoft-аккаунт не требуется.

## Что понадобится

- [Prism Launcher](https://prismlauncher.org/download/) 11.0.3 или новее;
- актуальная версия TLauncher;
- Java 17;
- 8–12 ГБ свободной оперативной памяти для игры;
- уникальный ник, согласованный с владельцем сервера.

## Шаг 1. Подготовить точные файлы через Prism

1. Скачайте файл вида
   `GregTech-Odyssey-Friends-0.5.6-v*.zip` со страницы
   [последнего релиза](https://github.com/ChiefSparrow/gregtech-odyssey-friends/releases/latest).
   Не скачивайте `Source code.zip`.
2. Не распаковывайте архив.
3. В Prism нажмите
   **Добавить сборку → Импорт → выбрать скачанный ZIP**.
4. Дождитесь окончания загрузки всех зависимостей.
5. Не запускайте Minecraft в созданном экземпляре.
6. Нажмите ПКМ по экземпляру → **Папка Minecraft**.
7. В открывшейся папке выполните:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\VERIFY-CLIENT.ps1
```

Продолжайте только после:

```text
CLIENT VERIFICATION PASSED
```

Если проверка сообщает `MISSING`, `SIZE`, `SHA256` или `EXTRA`, удалите этот
экземпляр Prism и импортируйте релиз заново. Не ищите отсутствующий мод по
названию в каталоге TLauncher.

## Шаг 2. Создать отдельный профиль TLauncher

1. Откройте TLauncher и нажмите **TL MODS**.
2. Нажмите **Создать** и задайте новое имя, например
   `GTO-Friends-1.0.1`.
3. Выберите строго:
   - Minecraft `1.20.1`;
   - загрузчик `Forge`;
   - Forge `47.4.20`.
4. Если TLauncher не предлагает именно Forge `47.4.20`, остановитесь: другая
   версия Forge для этой сборки не поддерживается.
5. Выберите созданный профиль. Если его игровая папка ещё не появилась,
   запустите пустой профиль один раз до главного меню и сразу закройте игру.
6. Через кнопку папки или меню профиля откройте **игровую папку именно этого
   модпака**. Обычно она находится внутри
   `.minecraft\versions\<имя модпака>\`, но ориентируйтесь на кнопку TLauncher.
7. Полностью закройте Minecraft, Prism и TLauncher.

Не используйте общую папку `.minecraft`: сборка должна находиться в отдельном
профиле, иначе старые моды попадут под проверку как `EXTRA`.

## Шаг 3. Перенести проверенный экземпляр

1. Снова откройте **Папку Minecraft** подготовленного экземпляра Prism.
2. Скопируйте всё содержимое этой папки в игровую папку созданного профиля
   TLauncher с заменой совпадающих файлов.
3. Не переименовывайте файлы в `mods`. Суффикс `.disabled` у 15 модов является
   частью эталонной конфигурации.
4. В папке профиля TLauncher повторно выполните:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\VERIFY-CLIENT.ps1
```

Запускайте игру только после повторного сообщения
`CLIENT VERIFICATION PASSED`.

## Шаг 4. Настроить запуск

В настройках созданного профиля TLauncher укажите:

- Java `17`;
- максимальную память `8192–12288 MiB`, рекомендуется `10240 MiB`;
- свой постоянный уникальный ник;
- Minecraft `1.20.1`;
- Forge `47.4.20`.

Не используйте кнопку автоматического обновления модов и не устанавливайте
дополнительные моды из `TL MODS`.

## Подключение к серверу

Аккаунт без лицензии не сможет войти на сервер с `online-mode=true`.
Чтобы допустить такие подключения, владелец сервера должен отдельно:

- перевести сервер в `online-mode=false`;
- установить совместимую с Forge систему регистрации/входа;
- включить белый список;
- защитить ники администраторов и игроков от подмены.

Один только белый список не защищает offline-сервер: без дополнительной
аутентификации другой человек может выбрать чужой ник. Не открывайте такой
сервер в интернет до настройки защиты.

## Обновление сборки

Для каждой новой версии повторите процедуру с новым чистым экземпляром Prism и
новым профилем TLauncher. Не обновляйте отдельные моды поверх старой версии.

Официальная справка TLauncher подтверждает, что Forge-моды устанавливаются в
папку `mods`, а отдельные модпаки имеют собственные игровые папки:

- [ручная установка Forge-модов](https://tlauncher.org/en/install-mods.html);
- [система модпаков и ручная установка](https://tlauncher.org/en/news_26/modpack-system_5100.html);
- [расположение папок модпаков](https://tlauncher.org/en/where-to-find-mc-saves-folder.html).
