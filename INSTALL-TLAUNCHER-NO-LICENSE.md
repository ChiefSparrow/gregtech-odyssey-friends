# Простая установка через TLauncher

Для этого варианта **Prism Launcher не нужен**. Готовый установщик сам создаёт
чистую игровую папку, скачивает строго зафиксированные версии модов и проверяет
каждую загрузку и все обязательные файлы по размеру и SHA-256.

> [!WARNING]
> TLauncher не является официальным лаунчером Mojang/Microsoft. Рекомендуемый
> вариант — лицензионная Minecraft: Java Edition и Prism Launcher. Пользователь
> сам отвечает за законность и безопасность стороннего лаунчера.

## Установка

1. Установите TLauncher, запустите через него любую версию Minecraft один раз,
   затем полностью закройте игру и лаунчер. Это нужно, чтобы TLauncher установил
   свою Java.
2. Скачайте
   [`GTO-Friends-TLauncher-Installer-v1.0.2.zip`](https://github.com/ChiefSparrow/gregtech-odyssey-friends/releases/latest/download/GTO-Friends-TLauncher-Installer-v1.0.2.zip).
   Не скачивайте `Source code.zip`.
3. Полностью распакуйте ZIP в обычную папку. Не запускайте файл прямо из окна
   архиватора.
4. Дважды нажмите `INSTALL-GTO-TLAUNCHER.bat`.
5. В открывшемся окне нажмите **Установить сборку** и дождитесь сообщения
   **Сборка установлена**.
6. Запомните показанную установщиком игровую папку.
7. Откройте настройки TLauncher и укажите:

   - игровая папка — именно та, которую показал установщик;
   - Minecraft — `1.20.1`;
   - Forge — строго `47.4.20`, вариант **без значка TL**;
   - Java — `17`;
   - память — `8192–12288 МБ`, рекомендуется `10240 МБ`.

8. Укажите свой постоянный уникальный ник и запустите игру.

Не устанавливайте поверх этой папки моды из `TL MODS` и не обновляйте отдельные
моды: это может сделать клиент несовместимым с сервером.

Значок **TL** включает систему скинов TLauncher, которая может автоматически
добавить в `mods` собственный `TLSkinCape`. В этой редакции он не проверен и
поэтому отключён намеренно. Если точной версии Forge `47.4.20` без значка TL в
списке нет, не выбирайте другую версию и сообщите владельцу сборки.

## Что делает установщик

Установщик не требует Prism, прав администратора и изменения системных
настроек. Он:

- создаёт отдельную игровую папку, если обычная `.minecraft` уже занята;
- распаковывает конфигурацию Friends Edition;
- скачивает 171 файл по HTTPS с официального CDN CurseForge;
- проверяет размер и SHA-256 каждой загрузки;
- проверяет все 177 обязательных файлов готового клиента;
- прекращает установку при любом несовпадении.

В архиве нет самодельного `.exe`. Запуск выполняет короткий читаемый
`INSTALL-GTO-TLAUNCHER.bat`, а сам установщик написан на Java. Его исходный код,
полный список URL и контрольных сумм лежат рядом в папке `SOURCE`.

## Антивирус и проверка файла

Установщик не запрашивает повышение прав, не меняет реестр, не отключает
антивирус, не использует PowerShell и не запускает загруженные моды во время
установки. Microsoft Defender проверяется перед публикацией релиза.

Windows или браузер всё равно могут показать обычное предупреждение о файле,
скачанном из интернета. Если антивирус именно обнаружил угрозу, **не отключайте
его и не добавляйте файл в исключения**. Удалите файл, скачайте его повторно
только из Releases и сравните SHA-256 с файлом `.sha256` рядом с архивом.

## Если установщик не открывается

- Убедитесь, что ZIP полностью распакован.
- Запустите любую версию Minecraft через TLauncher один раз и закройте её,
  затем повторите запуск `INSTALL-GTO-TLAUNCHER.bat`.
- Если TLauncher не предлагает Forge `47.4.20` без значка TL, не выбирайте
  похожую версию: сообщите владельцу сборки.
- Если установка прервалась из-за сети, просто запустите BAT повторно.

## Подключение без лицензии

Аккаунт без лицензии не войдёт на сервер с `online-mode=true`. Сервер,
разрешающий offline-аккаунты, должен иметь отдельную систему регистрации,
защиту ников и белый список. Один белый список не защищает от человека,
выбравшего чужой ник.

## Обновление

Для новой версии скачайте новый установщик из
[Releases](https://github.com/ChiefSparrow/gregtech-odyssey-friends/releases)
и установите её в новую папку. Не смешивайте файлы разных версий. Одиночные
миры переносите только после создания резервной копии папки `saves`.
